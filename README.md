# Angry Birds - Web Game

A browser-based Angry Birds clone built with HTML5 Canvas and Matter.js physics engine.

Play online: https://angellei10112358.github.io/angry_birds_no_comments/

## How to Play

- Drag the bird on the slingshot and release to launch
- Aim to hit the pigs and destroy their structures
- Clear all pigs to complete the level
- Each bird type has a unique special ability (tap while flying)

## Bird Types

- **Red**: Standard bird, no special ability
- **Yellow**: Tap to speed boost, effective against wood
- **Blue**: Tap to split into 3, effective against ice
- **Black**: Tap to explode, effective against stone
- **White**: Tap to drop an egg bomb and bounce upward

## Technical

- HTML5 Canvas rendering with camera system
- Matter.js for physics simulation
- Web Audio API for procedural sound effects
- No external dependencies beyond Matter.js CDN
- Responsive design with letterbox scaling
